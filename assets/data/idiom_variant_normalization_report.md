## 方法

1. 使用 `idiom_reversible_pair` 建立倒装/异形成语组。
2. 每组选择 `difficulty_original` 更低的一项作为 `canonical_word`；若相同，再按上一版难度、特征分和词面排序。
3. 组内 `semantic_difficulty` 取原始 LLM 分的最小值，表示语义难度。
4. 非标准/倒装形式使用有限 `surface_penalty`，公式为 `min(4, max(1, 0.25 * 原始分差))`；标准形式惩罚为 0。
5. 全局排序时让同组成员相邻，避免在 42 分等大桶内被分位映射拉开。
6. 最终 `difficulty` 仍为 1-50；同组最终分差被控制在 0-4。

## 新增/更新字段

| 0                                      | 1                                            |
|:---------------------------------------|:---------------------------------------------|
| difficulty_rebalanced_v1               | 上一版重平衡难度，处理前的 `difficulty` 备份 |
| difficulty                             | 本次处理后的最终 1-50 游戏难度               |
| variant_group_id                       | 倒装/异形成语组 ID；0 表示非倒装组           |
| canonical_word                         | 组内标准/主形式；非倒装词等于自身            |
| is_canonical                           | 是否标准形式，1 是，0 否                     |
| semantic_difficulty                    | 组内共享的语义难度，通常取组内最低原始分     |
| surface_penalty                        | 倒装/罕见形式惩罚，0-4                       |
| surface_difficulty_score               | `semantic_difficulty + surface_penalty`      |
| difficulty_base_before_variant_penalty | 施加形态惩罚前的分位难度                     |
