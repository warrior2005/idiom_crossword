# PRD.md — 成语填字游戏

> 产品需求文档 | 2026-06-14 | v1.0

---

## 一、产品概述

### 1.1 产品定位

一款面向中文用户的成语交叉填字手游。玩家在横纵交错的网格中，根据提示将正确的汉字填入空格，完成成语拼写。不同于传统的"看释义猜词"玩法，本作的核心乐趣在于**交叉推理**——一个汉字同时属于两个成语，必须满足两条线索才能确定答案。

### 1.2 核心体验

> "这个字既要是『画蛇添足』的最后一字，又要是『足智多謀』的第一字——那只能是『足』。"

交叉约束带来的**顿悟感**是其他字谜游戏无法替代的。认知心理学中称为"收敛性满足"：多重线索汇聚到唯一答案时，大脑的奖赏回路会给出比单纯识别更强的正向反馈。

### 1.3 参考产品

| 产品 | 长处 | 不足 |
|------|------|------|
| 成语小秀才 | 交叉填字核心玩法成熟，关卡量大 | 难度系统粗糙，偏重死记硬背 |
| Word Crossy | 交互流畅，动画精致 | 英文词库，不适用中文 |
| 成语玩命猜 | 画图猜词，趣味性强 | 缺乏交叉推理机制 |

**我们的差异化**：以交叉推理为核心，以精确的难度分级为壁垒，让不同语文水平的用户都能找到心流区间。

- 字谜游戏干扰选项太弱，一眼就能排除

## 三、核心玩法

### 3.1 基本规则

1. 网格由若干条成语交叉构成，横纵交替
2. 部分格子已填入汉字（起始提示），其余为空
3. 玩家点击空格，从候选字中选出正确的一个
4. 所有空格填满且全部成语正确即通关

### 3.2 交叉类型

```
例：3 条成语构成的网格

一 鸣 惊 人
        山
        人
        海 阔 天 空
```

对应成语：一鸣惊人（横向）、人山人海（纵向）、海阔天空（横向）。
交叉点"人"属于"一鸣惊人"和"人山人海"，交叉点"海"属于"人山人海"和"海阔天空"。
"人山人海"有两个"人"字，增加了额外迷惑性。交叉约束意味着玩家必须同时满足两条线索才能确定答案。

### 3.3 干扰字设计

每个空格提供 3-5 个候选字，包括：
- 正确答案
- 形近字（如候选"籍"vs 正确答案"藉"）
- 音近字（如候选"世"vs 正确答案"适"）
- 同部首字（如部首都是"扌"的字）

干扰字必须有迷惑性，不能让玩家仅凭排除法就能轻易过关。

### 3.4 提示系统

| 级别 | 功能 | 每关限制 |
|------|------|----------|
| 一字提示 | 揭示单个空格答案 | 3 次 |
| 成语提示 | 揭示一条完整成语 | 1 次 |
| 全图揭示 | 展示全部答案 | 视为放弃，不计入通关 |

## 四、难度体系

### 4.1 难度分级

采用 monotonic_quantile_v1 方法的 1-50 分制，全部 29,502 条四字成语按全局百分位等量映射。
每档约 590 条，分布均匀，避免传统评分的锚定坍缩问题。

| 段位 | 分数 | 代表成语（当前数据抽样）| 适合人群 |
|------|------|------|----------|
| 入门 | 1-10 | 自言自语、春暖花开、亡羊补牢 | 小学低年级 |
| 进阶 | 11-20 | 百里挑一、知己知彼、差强人意 | 小学高年级/初中 |
| 中等 | 21-30 | 力排众议、城下之盟、推本溯源 | 高中/大学 |
| 高阶 | 31-40 | 泛应曲当、窃据要津、玉骨冰肌 | 文学爱好者 |
| 大师 | 41-50 | 进退迍邅、杅穿皮蠹、林籁泉韵 | 古典文学研究者 |

### 4.2 关卡难度设计

- 每关选取 5-8 条成语组成网格
- 关卡难度 = 网格内成语难度分数的均值
- 允许 ±5 分浮动，保持同难度区间内成语风格一致
- 各段位首关作为"定级关"，帮助系统校准用户水平

### 4.3 难度递进

```
入门(10关) → 进阶(20关) → 中等(30关) → 高阶(40关) → 大师(50关)
                                                    ↓
                                              随机生成无限关卡
```

前四个段位为手工调优关卡，大师段位启用随机生成器，提供无限的挑战内容。

## 六、交互设计

### 6.1 填字流程

```
[关卡开始]
    ↓
[观看网格，起始字已填入]
    ↓
[点击空格] → 弹出候选字面板（3-5 字）
    ↓
[选择候选字] → 填入动画
    ↓
┌─ 全部填满？──→ [判断正误] ──→ •正确 → [过关动画] → [下一关]
│                                •错误 → [高亮错误字] → [重新填写]
└─ 未填满？────→ [继续]
```

### 6.2 网格渲染规范

- 每个汉字格 40×40 pt
- 空格的候选字面板在格子正下方弹出，不遮挡周边空格
- 横向成语从左到右，纵向成语从上到下
- 交叉点使用较深底色区分（比普通格深 10%）
- 已填字在交叉成语未完成前用半透明显示（表示"暂定"）

### 6.3 手势支持

| 手势 | 动作 |
|------|------|
| 点击空格 | 打开候选字面板 |
| 点击已填字 | 清除该字 |
| 长按空格 | 显示该字所属成语的提示 |
| 双指缩放 | 放大/缩小网格 |
