# PRD.md — 成语填字游戏

> 产品需求文档 | 2026-07-04 | v2.1

---

## 一、产品概述

### 1.1 产品定位

一款面向中文用户的成语交叉填字手游。玩家在横纵交错的网格中，根据提示将正确的汉字填入空格，完成成语拼写。不同于传统的"看释义猜词"玩法，本作的核心乐趣在于**交叉推理**——一个汉字同时属于两个成语，必须满足两条线索才能确定答案。

### 1.2 核心体验

> "这个字既要是『画蛇添足』的最后一字，又要是『足智多謀』的第一字——那只能是『足』。"

交叉约束带来的**顿悟感**是其他字谜游戏无法替代的。认知心理学中称为"收敛性满足"：多重线索汇聚到唯一答案时，大脑的奖赏回路会给出比单纯识别更强的正向反馈。

### 1.3 参考产品

| 产品 | 长处 | 不足 |
|------|------|------|
| 成语小秀才 | 交叉填字核心玩法成熟，关卡量大 | 难度系统粗糙，偏重死记硬背 |
| Word Crossy | 交互流畅，动画精致 | 英文词库，不适用中文 |
| 成语玩命猜 | 画图猜词，趣味性强 | 缺乏交叉推理机制 |

**我们的差异化**：以交叉推理为核心，以精确的难度分级为壁垒，让不同语文水平的用户都能找到心流区间。

## 二、目标用户

### 2.1 用户画像

| 画像 | 年龄 | 需求 | 使用场景 |
|------|------|------|----------|
| 小学生 | 8-12 | 趣味学成语，辅助语文学习 | 课后 15 分钟，家长陪同 |
| 中学生 | 13-18 | 巩固词汇量，应对考试 | 自习间隙，碎片时间 |
| 大学生/白领 | 19-35 | 消遣 + 轻度自我提升 | 通勤、午休、睡前 |
| 中老年文化爱好者 | 50+ | 防止认知衰退，文化娱乐 | 晨练后、午休 |

### 2.2 用户痛点

- 成语词典类 App 枯燥，缺乏互动
- 现有成语游戏难度要么太简单要么靠猜，缺乏梯度
- 想扩充词汇量但找不到有趣的方式
- 字谜游戏干扰选项太弱，一眼就能排除

## 三、核心玩法

### 3.1 基本规则

1. 网格由若干条成语交叉构成，横纵交替
2. 部分格子已填入汉字（起始提示），其余为空
3. 玩家点击空格，从候选字中选出正确的一个
4. 所有空格填满且全部成语正确即通关

### 3.2 交叉类型

```
例：3 条成语构成的网格

一 鸣 惊 人
        山
        人
        海 阔 天 空
```

对应成语：一鸣惊人（横向）、人山人海（纵向）、海阔天空（横向）。
交叉点"人"属于"一鸣惊人"和"人山人海"，交叉点"海"属于"人山人海"和"海阔天空"。
"人山人海"有两个"人"字，增加了额外迷惑性。交叉约束意味着玩家必须同时满足两条线索才能确定答案。

### 3.3 干扰字设计

每个空格提供 3-5 个候选字，包括：
- 正确答案
- 形近字（如候选"籍"vs 正确答案"藉"）
- 音近字（如候选"世"vs 正确答案"适"）
- 同部首字（如部首都是"扌"的字）

干扰字必须有迷惑性，不能让玩家仅凭排除法就能轻易过关。

### 3.4 提示系统

| 级别 | 功能 | 每关限制 |
|------|------|----------|
| 一字提示 | 揭示单个空格答案 | 3 次 |
| 成语提示 | 揭示一条完整成语 | 1 次 |
| 全图揭示 | 展示全部答案 | 视为放弃，不计入通关 |

## 四、难度体系与成长系统

### 4.1 成语难度评分

采用 1-50 分制，全部 29,502 条四字成语按全局百分位等量映射，每档约 590 条。

### 4.2 成长系统概要

采用**科举仕途**主题，共 20 级（童生→位极人臣），暴雪式指数经验曲线，先易后难。
玩家通过通关获取经验，升级解锁称号和道具奖励。

> 完整设计（等级表、经验公式、螺旋难度模型、道具系统）详见：
> `docs/superpowers/specs/2026-07-04-growth-system-design.md`

### 4.3 关卡结构

- **总关卡**：10,000+ 关，螺旋难度递增
- **教学关**（1-5关）：每关 5 条成语
- **正式关**（6关+）：每关 8-12 条成语
- **成语保底**：每条成语至少出现 3 次

## 六、交互设计

### 6.1 填字流程

```
[关卡开始]
    ↓
[观看网格，起始字已填入]
    ↓
[点击空格] → 弹出候选字面板（3-5 字）
    ↓
[选择候选字] → 填入动画
    ↓
┌─ 全部填满？──→ [判断正误] ──→ •正确 → [过关动画] → [下一关]
│                                •错误 → [高亮错误字] → [重新填写]
└─ 未填满？────→ [继续]
```

### 6.2 网格渲染规范

- 每个汉字格 40×40 pt
- 空格的候选字面板在格子正下方弹出，不遮挡周边空格
- 横向成语从左到右，纵向成语从上到下
- 交叉点使用较深底色区分（比普通格深 10%）
- 已填字在交叉成语未完成前用半透明显示（表示"暂定"）

### 6.3 手势支持

| 手势 | 动作 |
|------|------|
| 点击空格 | 打开候选字面板 |
| 点击已填字 | 清除该字 |
| 长按空格 | 显示该字所属成语的提示 |
| 双指缩放 | 放大/缩小网格 |

## 八、商业模式

### 8.1 收入模型

| 方式 | 说明 |
|------|------|
| 基础关卡免费 | 全部 10,000+ 关免费游玩 |
| 功能道具内购 | 提示卡×10 / 复活卡×5 等组合包 |
| 装饰道具内购 | 稀有皮肤、限定头像框、称号特效 |
| 去广告 | ¥3，移除所有广告 |

### 8.2 内购设计

- **内购入口始终存在**，玩家可随时访问商城
- **功能道具包**：提示卡×10 / 复活卡×5 等组合包
- **装饰道具**：等级奖励以外的稀有皮肤、限定头像框、特效
- **不卖数值**：内购不影响关卡难度和成语选择

### 8.3 广告策略

- 基础关卡每 3 关展示一次插屏广告
- 不使用激励视频广告（破坏游戏节奏）
